# Welcome to the Blog App

Welcome to your Next.js blog demo. This app is built with modern Next.js features and includes a simple blog browsing experience, user authentication, reading lists, and API token management.

## What you can do here

- Browse published blog posts on the `/blogs` page
- Sign in with credentials from the `/login` page
- View your profile and reading list on `/me`
- Track unread and read blogs
- Generate or view an API token for your account

## App highlights

- Server components for fast page rendering
- Client-side login flow using `next-auth`
- Dynamic blog listing and filtering
- User-specific reading list management
- Clean Tailwind-inspired styling with responsive layouts
